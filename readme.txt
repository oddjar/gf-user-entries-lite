# User Entries Lite for Gravity Forms

User Entries Lite for Gravity Forms is a WordPress plugin that displays the user's last 5 form entries in the user profile, visible only to administrators.

## Features

- Displays a list of the last 5 Gravity Forms entries for each user
- Entries are displayed in the WordPress User Profile
- Only visible to administrators
- Provides a quick overview of user activity within Gravity Forms

## Installation

1. Download the plugin ZIP file.
2. Log in to your WordPress admin panel and navigate to "Plugins" > "Add New".
3. Click on the "Upload Plugin" button and select the downloaded ZIP file.
4. Click on the "Install Now" button and wait for the installation to complete.
5. Once installed, click on the "Activate Plugin" button to activate the plugin.

## Usage

1. After activating the plugin, navigate to the WordPress User Profile of any user.
2. Scroll down to the "User Entries from Gravity Forms" section.
3. The section will display a table with the last 5 Gravity Forms entries submitted by the user.
4. The table includes the form title, entry date, and a link to view the entry details.
5. Clicking on the "View Entry" link will open the entry details page in a new tab.

Note: The "User Entries from Gravity Forms" section is only visible to administrators.
